Информация о пакете для скачивания
Свою информацию вы можете найти в файлах HTML из этого пакета для скачивания. Для их просмотра мы рекомендуем использовать ПК. По ссылкам из файлов вы можете перейти к своей информации. Ссылки не открываются на мобильных устройствах.
На мобильном устройстве воспользуйтесь папками из пакета для скачивания, чтобы найти файлы HTML. Информация определенного типа хранится в отдельном файле HTML. Для ее просмотра нужно открыть файл.
